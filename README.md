# Ubertitle

**Professionelle Übertitel-Software für Theater & Oper**

Ubertitle ist eine Desktop-Applikation für die Live-Steuerung von Übertiteln bei Opern- und Theateraufführungen. Entwickelt für den täglichen Produktionsbetrieb — zuverlässig, schnell, auf den Punkt.

---

## Features

- **Live-Steuerung** — manuelle und zeitgesteuerte Übertitel-Ausgabe in Echtzeit
- **Multi-Display** — separate Steuer- und Ausgabe-Fenster, ideal für Bühnenregie
- **WebSocket-Sync** — mehrere Clients können gleichzeitig den Output empfangen
- **OSC-Integration** — Anbindung an Licht- und Tontechnik-Systeme (Qlab, MA Lighting etc.)
- **Import** — strukturierter Import von Texten aus Standardformaten
- **Tastaturoptimiert** — schnelle Bedienung auch unter Live-Druck
